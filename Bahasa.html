Belajar Conditional Sentences + Quiz Interaktif

Zero Conditional

Zero Conditional digunakan untuk menyatakan fakta umum, kebenaran ilmiah, atau kejadian yang selalu terjadi jika kondisi tertentu terpenuhi.

Rumus: If + Present Simple, Present Simple

Contoh 1: If you heat water, it boils.

Artinya: Jika kamu memanaskan air, air itu mendidih.

Contoh 2: If you drop an object, it falls.

Artinya: Jika kamu menjatuhkan suatu benda, benda itu jatuh.

Type 1

Type 1 digunakan untuk kemungkinan nyata di masa sekarang atau masa depan. Biasanya dipakai untuk prediksi atau janji yang mungkin terjadi.

Rumus: If + Present Simple, Will + Verb

Contoh 1: If it rains, I will stay home.

Artinya: Jika hujan, saya akan tetap di rumah.

Contoh 2: If I pass the exam, I will celebrate tonight.

Artinya: Jika saya lulus ujian, saya akan merayakannya malam ini.

Type 2

Type 2 digunakan untuk situasi pengandaian yang tidak nyata atau tidak mungkin terjadi saat ini atau di masa depan.

Rumus: If + Past Simple, Would + Verb

Contoh 1: If I had money, I would buy a car.

Artinya: Jika saya punya uang, saya akan membeli mobil.

Contoh 2: If I were taller, I would play basketball.

Artinya: Jika saya lebih tinggi, saya akan bermain basket.

Type 3

Type 3 digunakan untuk pengandaian di masa lalu, menyatakan sesuatu yang tidak terjadi dan akibatnya juga tidak terjadi.

Rumus: If + Past Perfect, Would have + Verb

Contoh 1: She would have finished the exam if she had had more time.

Artinya: Dia akan menyelesaikan ujian jika dia memiliki lebih banyak waktu.

Contoh 2: If I had studied harder, I would have passed the test.

Artinya: Jika saya belajar lebih giat, saya akan lulus ujian.
